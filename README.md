# open_router.ipynb
- Бизнес-задача: классифицировать товары маркетплейса по спортивным категориям.
- На входе используются названия товаров и item_id из тестового CSV.
- Результат нужен для подготовки файла с предсказанными категориями.
- Для работы с данными применялись pandas, json, re и tqdm.
- Для LLM использован OpenAI-compatible клиент через локальный Ollama/OpenRouter endpoint.
- Модель в ноутбуке: qwen2.5:7b.
- Подход: двухэтапный prompt engineering с рассуждением и строгим JSON-ответом.
